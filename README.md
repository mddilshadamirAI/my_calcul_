# Dilshad's Calc Pro
An advanced, mobile-optimized glassmorphic utility web application featuring dual scientific operational modes, persistent localized math logging, native text-to-speech accessibility, and a decoupled private media storage vault. 

This project showcases modern front-end engineering, local memory architecture, and fluent responsive layouts optimized for standalone mobile application performance.

---

## 🚀 Advanced Features

* **🔬 Expandable Scientific Panel:** Seamlessly toggles from a sleek standard interface into a high-powered scientific matrix handling advanced operations like trigonometry (`sin`, `cos`, `tan`), logarithmic functions (`log`, `ln`), custom exponents (`a^x`, `e^x`), factorials (`x!`), constants (`π`, `e`), and complete nested parenthesis tracking.
* **📜 Persistent 100-Calculation History Log:** Engineered a localized database queue that tracks and displays your last 100 successful equations. Clicking any item in the history log fluidly loads that historic result right back into live operational memory. 
* **🔊 Acoustic Review Engine (Revise Mode):** Integrated with the native **Web Speech API** to dynamically map standard equations and functional terms into clear phonetic strings, reading back complex math operations aloud to improve overall accessibility.
* **🔒 Hidden Security Media Vault:** A completely stealthy locker hidden directly beneath the calculator UI. Entering your custom passcode and double-tapping the `=` sign bypasses the calculation matrix to safely open a localized media vault where you can upload, view, and store personal images or videos.
* **📱 Progressive Web App (PWA) Integration:** Built with dedicated service metadata configurations allowing users to install the web link directly onto an Android or iOS home screen as a standalone application complete with a custom app icon and full-screen display.

---

## 💻 Tech Stack

* **HTML5 & Vanilla ES6+ JavaScript:** Advanced procedural logic, custom math compilers, array matrix handling (`.unshift()`, `.pop()`), and native browser execution.
* **CSS3 Glassmorphic Styling:** Implements structural flexbox/grid layout design, custom dark-mode aesthetics, and hardware-accelerated transitions (`backdrop-filter`) built to fit standard mobile browser viewports natively.
* **Web Storage API (`localStorage`):** Used to securely manage persistent local state arrays for both the 100-record calculation log and the hidden media files, surviving complete device reboots.
* **Web Speech API:** High-efficiency browser speech synthesis framework.

## 🤝 Project Transparency & Development Methodology
In alignment with modern software development standards, **this application was engineered utilizing advanced AI-assisted pair-programming.** While the structural features, design flow, architectural requirements, and logical features (combining accessibility features, history logs, and a hidden media system) were entirely driven by the creator, the code optimization, complex mathematical token parsing, and edge-case mobile scaling were generated in partnership with AI. This workflow represents the highly productive, modern "indie-hacker" approach to building software rapidly.

## 📱 How to Install on Mobile

You can run this application independently as a native-feeling standalone app on your smartphone without downloading it from an app store:

1. **Open the Live Link:** Navigate to your hosted **GitHub Pages** URL inside your mobile phone browser.
2. **Add to Home Screen:**
   * **On iOS (Safari):** Tap the **Share** menu button at the bottom of the screen and select **"Add to Home Screen"**.
   * **On Android (Chrome):** Tap the **Three Dots** menu icon in the top right corner and select **"Install App"** or **"Add to Home Screen"**.
3. **Launch:** Open the app directly from your mobile home screen grid. It will launch with its own icon, a hidden browser address bar, and an optimized touch canvas.
