# 🧮 DILSHAD'S CALCULATOR PRO
### Next-Gen Scientific Computing | PWA-Ready | Secure Media Vault

[**Launch Live App**](https://mddilshadamirai.github.io/my_calcul_/)

### 🌐 OVERVIEW
**Dilshad's Calc Pro** is a high-performance, glassmorphic utility application designed for mobile-first precision. It transcends the capabilities of a standard calculator by integrating scientific computation, persistent data logging, and a stealth-mode media vault—all within a responsive, PWA-enabled interface.

---

### 🚀 ADVANCED FUNCTIONALITY

*   **🔬 Scientific Matrix:** Seamlessly toggle between standard arithmetic and advanced scientific operations, including trigonometric functions (`sin`, `cos`, `tan`), logarithms, exponents, and complex parenthesis nesting.
*   **📜 Persistent Memory Queue:** Features an engineered localized database that tracks your last 100 calculations, allowing for instant recall of historic results.
*   **🔊 Acoustic Accessibility:** Integrated **Web Speech API** translates complex equations into clear audio, ensuring math operations are accessible and easy to review.
*   **🔒 Stealth Media Vault:** A security-first feature hidden beneath the calculator UI. Access your private media storage through a custom passcode and unique gesture-based triggering.
*   **📱 PWA-Ready:** Optimized for installation as a native-feeling standalone application on both Android and iOS devices.

### 📱 INSTALLATION GUIDE (MOBILE)
Run this application as a standalone native app on your device:

1.  **Open:** Navigate to the **Live Link** in your mobile browser.
2.  **Add to Home Screen:**
    *   **iOS (Safari):** Tap *Share* → Select *"Add to Home Screen"*.
    *   **Android (Chrome):** Tap *Three Dots* → Select *"Install App"* or *"Add to Home Screen"*.
3.  **Launch:** Experience a seamless, full-screen interface directly from your app grid.
