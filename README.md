## Calculator App
A lightweight, responsive web-based calculator built with vanilla JavaScript and HTML. This project is optimized for both desktop browsers and mobile devices.

## 🚀 Features
Real-time Arithmetic: Handles addition, subtraction, multiplication, and division with instant precision.

Mobile-Ready: Designed with a responsive UI that adapts to any screen size.

Web-to-App: Fully functional as a Progressive Web App (PWA) that can be installed on mobile home screens.

## 💻 Tech Stack
HTML5: Semantic structure.

CSS3: Modern, mobile-responsive layout.

JavaScript: Core logic and DOM manipulation.

GitHub Pages: Automated cloud deployment.

## 📱 How to Install on Mobile
You can use this calculator as a standalone app on your phone without downloading from an app store:

Open the Live Link: Open the GitHub Pages URL in your mobile browser.

Add to Home Screen:

iOS (Safari): Tap the Share icon and select "Add to Home Screen".

Android (Chrome): Tap the Three Dots and select "Install App" or "Add to Home Screen".

Launch: Open it directly from your app drawer
